# Quick Revision Cheat Sheet

## 60-second refresh

- SQL: joins, group by, window functions
- ETL: extract-transform-load, dedup, incremental
- Spark: lazy eval, shuffles, partitioning, caching
- Airflow: DAG, retries, XCom, monitoring
- Kafka: topic, producer/consumer, offsets
- Cloud: S3 (lake), EC2 (compute), RDS (db)
- System design: source -> ingest -> process -> store -> orchestrate

## Pre-interview plan

1. Revise all `interview_prep/*.md`
2. Practice `09_mock_questions.md`
3. Explain your capstone in 60-90 seconds

"I built an end-to-end data platform with batch + streaming ingestion, orchestration, cloud storage, and reliability controls (validation, logging, monitoring)."
