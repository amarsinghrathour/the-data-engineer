# ETL Interview Prep

Q: What is ETL?
A:
Extract -> Transform -> Load

---

Q: ETL vs ELT?
A:
- ETL -> transform before load
- ELT -> load first, transform later

---

Q: Challenges in ETL?
A:
- Data quality issues
- Schema changes
- Large data volume
- Failures/retries

---

Q: How do you handle duplicates?
A:
- Primary keys
- Deduplication logic
- Upserts

---

Q: What is incremental processing?
A:
Processing only new/updated data instead of full dataset
