## Sample prompts

Use these for timed practice; tie answers back to projects in `projects/`.

1. **Design Uber’s data pipeline** (or similar ride-hailing / mobility domain): events, batch aggregates, SLAs, privacy boundaries.
2. **Late-arriving data:** watermarking vs reprocessing; idempotent sinks; downstream contract.
3. **Optimize a slow Spark job:** skew, partitions, shuffle, broadcast joins, spill, cluster sizing.

Add flashcards, SQL drills, and mock notes under this folder as you build them.
