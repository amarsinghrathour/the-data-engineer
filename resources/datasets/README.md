# Shared datasets

Reusable fixtures for local development and notebooks.

Primary files:

- `users.json`
- `orders.json`
- `events.json`
- `dirty_users.json`
- `transactions.csv`
- `large_orders.csv`

Generated files (optional):

- `generated_users.json`
- `generated_events.json`

Generate larger samples with:

```bash
python3 resources/scripts/generate_data.py
```
