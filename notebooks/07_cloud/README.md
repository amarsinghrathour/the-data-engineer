# Phase 7 — Cloud

**Weeks 21–23**

Learn how to deploy data pipelines in cloud environments, focusing on AWS-first concepts that transfer to Azure and GCP.

**Project:** cloud-based pipeline (`05_cloud_project.ipynb`).

---

## Cloud Strategy

Pick one cloud and go deep first:

- AWS (recommended for this course)
- Azure
- GCP

The notebooks assume AWS naming (`S3`, `EC2`, `RDS`), but architecture patterns apply across providers.

---

## Notebooks (in order)

| File | Topic |
|------|--------|
| `01_cloud_basics.ipynb` | Cloud fundamentals for DE (AWS) |
| `02_storage_and_data_lake.ipynb` | S3 data lake architecture and storage patterns |
| `03_compute_and_pipeline_deploy.ipynb` | EC2 deployment, cron automation, cloud ops |
| `04_cloud_pipeline.ipynb` | S3 + EC2 + PostgreSQL integrated pipeline |
| `05_cloud_project.ipynb` | Portfolio project: cloud-native data platform |
